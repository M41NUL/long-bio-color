## Usage

1. Open the tool in your browser
2. Select a color from the palette or use the custom color picker
3. Type your bio in the editor
4. Apply formatting using **B**, _I_, U buttons
5. Check the live preview
6. Click **Copy Bio** and paste it in Free Fire

---

## Color Tag Format

```
[FF0000]Red Text
[00FF00]Green Text
[FFFF00]Yellow Text
[b]Bold Text
[i]Italic Text
[u]Underline Text
```
